# Machine Learning Journey

An evolving collection of machine learning concepts implemented from scratch in Python/NumPy — building each algorithm's math manually before relying on libraries, to develop a genuine understanding of what's happening under the hood rather than just calling `.fit()`.

## Approach

Each topic follows the same process:
1. Work through the underlying math by hand
2. Implement it from scratch using NumPy
3. Validate the result against standard library implementations (e.g., scikit-learn) for correctness
4. Visualize where it helps build intuition (cost surfaces, decision boundaries, convergence behavior, etc.)

The goal isn't just working code — it's understanding *why* each piece works, including the debugging process when something doesn't.

## Structure

Each folder/notebook corresponds to a specific concept or algorithm, generally covering the theory, a from-scratch implementation, and a comparison against library-based results.

## Status

This repository is updated continuously as I progress through my machine learning learning path — spanning regression, classification, and related techniques.
